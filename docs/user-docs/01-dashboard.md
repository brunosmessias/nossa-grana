# 🏠 Dashboard

> O Dashboard é a tela principal do sistema, onde você vê um resumo rápido de tudo que está acontecendo nas finanças da família.

## Visão Geral

O Dashboard funciona como um "hub" — ele mostra informações rápidas e Links para as páginas dedicadas onde você pode ver mais detalhes. É como a tela inicial do seu banco, mas para toda a família.

Você encontra nele:
- O saldo total e saldo do mês
- Cards rápidos mostrando quantas contas, categorias, transações e membros existem
- Botões para criar contas, categorias e registrar transações
- As transações do mês separadas em despesas e receitas

## Como Funciona

### Navegação por mês

No topo do dashboard, você pode navegar entre meses usando as setas ◀️ ▶️. Isso muda os dados mostrados para o mês selecionado.

### Resumo financeiro

Dois cards grandes mostram:
- **Saldo do mês**: receitas menos despesas do mês selecionado
- **Saldo total**: soma de todas as contas ativas da família

### Cards de resumo

Quatro cards menores mostram rapidamente:
- Quantidade de contas ativas
- Quantidade de categorias cadastradas
- Quantidade de transações registradas
- Quantidade de membros na família

### Transações do mês

A seção principal mostra duas tabelas:
- **Despesas** — todos os gastos do mês, com total
- **Rendas** — todas as receitas do mês, com total

Cada tabela mostra data, descrição, categoria e valor.

## Regras Importantes

| Regra | Detalhe |
|-------|---------|
| Sem família, sem dashboard | Se o usuário não tem família, é redirecionado para criar uma |
| Sidebar de navegação | A barra lateral permite ir para qualquer módulo do sistema |
| Dados em tempo real | Os dados são atualizados toda vez que você cria algo novo |

## Perguntas Frequentes

**Posso personalizar o dashboard?**
Hoje o dashboard mostra informações fixas. Personalização está nos planos futuros.

**Os dados são atualizados automaticamente?**
Sim, sempre que você cria uma transação, conta ou categoria, os dados são atualizados.
